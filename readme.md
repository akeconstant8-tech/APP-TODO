# Todo App

## Description

Todo App est une application web simple permettant de gérer une liste de tâches.

## Fonctionnalités

* Ajouter une tâche
* Marquer une tâche comme terminée
* Supprimer une tâche
* Sauvegarde automatique dans le navigateur

## Technologies utilisées

* HTML
* CSS
* JavaScript

## Installation

1. Cloner le projet

```bash
git clone https://github.com/votre-utilisateur/todo-app.git
```

2. Ouvrir le fichier index.html dans un navigateur.

## Déploiement

L'application est déployée avec GitHub Pages.
